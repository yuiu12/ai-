微积分与最优化
最优化理论
    做事过程中，希望以最小的代价取得最大的收益。解决一些工程问题时，遇到多种因素交织在一起与决策目标相互影响的情况；我们会使用最优化数学理论来应对这一挑战，而大家了解的线性规划也是最早的最优化方法之一。
    机器学习 = 模型 + 策略 + 算法。公式中的算法指的就是优化算法。大家在算法求职面试过程中，在项目模型效果调优过程中，经常会遇到优化算法。

最优化问题的数学描述
    min  f(x) 
    s.t.  hi(x) = 0 
          gj(x) <= 0 
    设计变量:x是一个实数域范围内的n维向量，被称为决策变量或问题的解
    目标函数:f(x)维目标函数
    约束条件:hi(x) = 0称为等式约束，gj(x) <= 0为不等式约束,i,j=0,1,2,...

凸集与凸集分离定理
1.凸集
    实数域R上(或复数C上)的向量空间中，如果集合S中仍两点的连线上的点都在S内，称为集合S为凸集
2.超平面和半空间
    二维空间的超平面就是一条线，三维空间的超平面就是一个面。
    超平面: H={x 属于(R**n)|a1 + a2 + ...+ an=b}
    半空间: H+={x 属于 (R**n) | a1 + a2 + ... + an >= b} 
3.凸集分离定理
    两个凸集分离，两个凸集合没有交叉和重合的部分，用一张超平面将两者隔在两边
4.凸函数
    定义域在某个向量空间的凸子集C上的实值函数
    凸函数的几何含义是：
        函数任意两点x1和x2之间的部分，位于玹x1x2的下方或其曲线任一点切线上方
    如果一个函数是凸函数，局部最优点就是它的全局最优点。这个性质在机器学习算法优化中最重要的应用，机器学习模型最后就是在求某个函数的全局最优点
    一旦证明该函数是凸函数，我们只用求它的局部最优点了。

梯度下降算法
1.背景
2.梯度下降算法
    多元微分学中，梯度就是函数的导数方向。梯度法是求解无约束多元函数极值最早的数值方法，很多机器学习的常用算法都是以它作为算法框架，进行改进而导出更为复杂的优化方法。
    minf(x)----->求解目标约束f(x)的最小值
    沿梯度方向迭代求解误差的收敛速度最快
3.梯度下降法的过程
    初始化：选择一个初始点作为搜索的起点。这个起点可以是随机选择，也可以是根据问题的具体情况来确定。
    计算梯度：在当前点计算目标函数的梯度。梯度的方向指向函数值增加最快的方向，我们关注的是梯度的反方向
    更新参数：根据计算出的梯度和预设的学习率，更新参数。学习率决定了在梯度方向上前进的步长大小，他对算法的收敛速度和稳定性有很大影响
    判断终止条件  检查是否满足终止条件，如梯度的幅度是否小于某个阈值、迭代次数是否达到预设值或改进幅度是否足够小。如果满足终止条件，则停止迭代；否则，返回第二步继续迭代。
5.随机梯度下降算法
    步长值取的越大，收敛速度越快，但是带来的可能后果就是容易越过函数的最优点，导致发散；
    步长取太小，算法的收敛速度又会明显降低
    随机梯度下降法引进了随机样本抽取方式，提取了一种动态步长取值策略。又要优化精度，又要满足收敛速度
    随机梯度下降方法以损失很小的一部分精确度和增加一定数量的迭代次数为代价，换取总体的优化效率的提升，增加的迭代次数远远小于样本数量。
    批量梯度下降法每次迭代都会计算训练集中所有数据，而随机梯度下降法每次迭代只会随机取了训练集中的一部分样本数据进行梯度计算

牛顿法
1.牛顿法简介
    求解无约束最优化问题的常用方法，最大优点是收敛速度快。二阶收敛，梯度下降是一阶收敛，牛顿法更快
2.牛顿法的推导
3.牛顿法的过程
    初始化一个初始点 x0，并计算目标函数 f(x) 在 x0 处的值和梯度。
    计算梯度的模长，如果小于给定的容差 tol，则停止迭代。
    计算目标函数 f(x) 在 x0 处的一阶导数 df(x)。
    计算逆矩阵 df(x)^-1。
    计算梯度向量 grad = -df(x) * grad。
    更新当前点 x = x - grad。
    重复步骤2~6，直到满足条件或达到最大迭代次数。

阻尼牛顿法
1.背景
    原始牛顿法不能保证函数值稳定下降，严重时甚至造成序列发散而导致计算失败。阻尼牛顿法消除这一弊端。

拟牛顿法
1.概述
    牛顿法每一步要求解目标函数的Hessen矩阵，计算量比较大。
    通过正定矩阵近似代替Hessen矩阵的逆矩阵，就是拟牛顿法
